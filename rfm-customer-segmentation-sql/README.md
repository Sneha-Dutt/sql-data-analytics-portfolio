# Customer Segmentation using RFM Analysis (SQL)

## Project Overview
This project analyzes customer purchasing behavior using RFM analysis.

RFM stands for:
Recency
Frequency
Monetary

The goal is to segment customers based on purchasing patterns.

---

## Dataset Summary

Total customers analyzed: ~96,000

Average customer spending: R$167

Highest customer revenue: R$13,664

---

## Key Insights

Some customers have not purchased for more than 500 days indicating churn risk.

Most customers place only 1–2 orders showing reliance on first-time buyers.

High-value customers contribute disproportionately to revenue.

---

## Customer Segments

Champions  
Loyal Customers  
Potential Loyalists
