# Customer Retention & Cohort Analysis

## Project Overview
This project analyzes customer retention patterns using SQL cohort analysis.

The goal is to understand:
- customer purchasing behavior
- retention over time
- repeat customer trends

## Dataset
Olist Brazilian E-commerce Dataset

Dataset Link:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Tools Used
- SQL
- MySQL
- Data Analysis

## Tables Used
- customers
- orders
- order_items
- payments
